README.md🛠️ Technical Portfolio – Application Specialist Skills
Detta repo demonstrerar tekniska färdigheter inom fyra nyckelområden som är relevanta för rollen som Applikationsspecialist:
Område	Mapp	Tekniker
SQL – databashantering	`/sql-queries`	SQLite, SELECT, JOIN, aggregering, subqueries
API-integration & testning	`/api-testing`	Python requests, pytest, JSON-validering, felhantering
Scripts & batchjobbhantering	`/batch-scripts`	Python, schemaläggning, loggning, filhantering
Teknisk konfiguration	`/config-management`	YAML/JSON-config, miljövariabler, .env-filer
---
🗂️ Projektöversikt
1. SQL – Databashantering
Skriver SQL-queries mot en SQLite-databas med simulerad verksamhetsdata (kunder, ordrar, produkter). Visar förmågan att analysera, filtrera och aggregera data på ett strukturerat sätt.
2. API-testning
Strukturerade tester av ett externt REST API (JSONPlaceholder). Inkluderar statuskodvalidering, schema-kontroll och felscenarier – precis som en Applikationsspecialist gör i praktiken.
3. Batch-scripts
Automatiserade Python-scripts som körs schemalagt: filbearbetning, loggning och felhantering. Simulerar verkliga batchjobb i en integrationsmiljö.
4. Teknisk konfiguration
Hanterar applikationskonfiguration via YAML och `.env`-filer. Visar hur man separerar konfiguration från kod och tolkar tekniska integrationsspecifikationer.

🚀 Kom igång
```bash
# Klona repot
git clone https://github.com/DITT-ANVÄNDARNAMN/technical-portfolio.git
cd technical-portfolio

# Installera beroenden
pip install -r requirements.txt

# Kör alla tester
pytest api-testing/ -v
```
📋 Krav
Python 3.10+
pip (pakethanterare)
